# technique-tool
technique-tool, a command-line tool for mapping shader constants between Call of Duty: Modern Warfare 2 and Call of Duty: Online.

## Folder structure
technique-tool\
|- techniques\
|- technique-tool.exe

## Usage
Simply run ``./technique-tool.exe`` from your command-line.
